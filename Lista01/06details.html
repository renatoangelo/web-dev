Perguntas Frequentes (FAQ)

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web e aplicações web. Ela estrutura o conteúdo da página.

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML. Ela controla as cores, fontes e layout.

O que é JavaScript?

JavaScript é uma linguagem de programação que permite implementar itens complexos em páginas web, como interatividade, animações e atualizações de conteúdo.

O que é um repositório Git?

É uma pasta de projeto que o Git monitora. Ele contém todos os arquivos do projeto e o histórico completo de todas as alterações feitas neles.
